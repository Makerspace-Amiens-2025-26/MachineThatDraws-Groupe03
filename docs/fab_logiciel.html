4. Logiciel, Slicer & Interface (Software)

La mécanique et l'électronique sont prêtes. L'ultime étape du projet a été de concevoir une chaîne logicielle capable de piloter la machine. L'équipe a pris la décision radicale de tout développer "from scratch" en technologies Web (HTML/CSS/JS), éliminant ainsi le besoin de logiciels tiers lourds.

1. Le Cerveau Bas-Niveau : Le Firmware

La carte mère (Arduino Uno V1, puis ESP32 UNO) doit pouvoir comprendre les commandes de déplacement. Nous avons flashé le firmware GRBL Servo (une version de GRBL modifiée spécifiquement pour gérer un signal PWM sur l'axe Z au lieu d'un moteur classique).

  • Déploiement : Initialement flashé via l'IDE Arduino pour le POC, nous sommes ensuite passés sur l'environnement professionnel PlatformIO pour gérer plus facilement les bibliothèques et la compilation sur l'ESP32 UNO.

2. La Magie du Web : L'API Google Serial

Habituellement, les machines CNC nécessitent un logiciel installé sur l'ordinateur (comme Universal Gcode Sender) pour envoyer les commandes à la machine. Notre interface Web fait tout cela directement depuis le navigateur Chrome.

🌐 Communication Directe (Web Serial API) :
Grâce à l'API navigator.serial intégrée en JavaScript, notre site Web ouvre un port de communication (BaudRate 115200) directement avec le port USB du PC. Le navigateur lit l'état de la machine en temps réel et envoie le G-Code ligne par ligne, tout en gérant les accusés de réception (le fameux ok de GRBL).

3. Workflow Utilisateur : Comment lancer un dessin ?

L'objectif de cette interface était de rendre la machine utilisable par n'importe qui en quelques clics, sans connaissances techniques :

  1. Brancher le câble USB reliant la machine au PC.
  2. Cliquer sur "Connecter USB" sur notre interface Web et sélectionner le bon port série.
  3. Faire le Homing ($H) pour que la machine aille chercher ses capteurs de fin de course et trouve son origine physique.
  4. Importer une image (ou générer un paysage Topographique).
  5. Ajuster les réglages (Taille, Seuil de contraste, Flou) via les sliders avec un aperçu visuel en temps réel du plateau.
  6. Cliquer sur "Lancer l'impression" et admirer le résultat !
